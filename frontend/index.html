About AIWOCS

AIWOCS (AI Web Content Summarizer) is a mini group project built by 5 B.Tech students to make long content easier to consume. Our goal is simple: transform lengthy articles, notes, and documents into crisp summaries with key insights.

  • ⚡ Multiple input modes: manual text, webpage URL, and file upload
  • 🧠 AI abstractive summarization with readability metrics
  • 🎯 Keyword and important-point highlighting for quick understanding
  • 📥 Instant export to TXT or PDF for sharing and revision

Tech Stack

Frontend

Semantic HTML, responsive CSS, and vanilla JavaScript interactions.

Backend

Node.js + Express API for extraction, parsing, and summarization routing.

AI Model

Hugging Face inference using BART-large-CNN for concise abstractive summaries.
